## Agricultural-predictive-analytics

## Project Overview

End-to-end predictive analytics project using Python, feature engineering, and Linear Regression to predict crop yield.


**Author:** Anuoluwapo Emmanuel Oke — Aspiring Predictive Data Analyst 

Key highlights:

- Data cleaning & handling missing values  
- Feature engineering (Low Rainfall & Low Yield risk flags)  
- Exploratory Data Analysis (EDA) & correlation visualization  
- Predictive modeling & evaluation  
- Actionable insights for AgriTech stakeholders
